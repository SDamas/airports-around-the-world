AWW - Airports Around the World

Hello, welcome to AWW. A project made by Samad Damas.

This is a college project built on Javascript, with the intent to show how to fetch data from an external API. The data used is from docs.aviationAPI.com

This is a dynamic webpage, and display data according to users interaction.
